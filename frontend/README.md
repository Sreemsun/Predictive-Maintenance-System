# Frontend

This folder contains all frontend assets for the Predictive Maintenance System.

## Contents

### Static Assets (`static/`)

#### CSS Stylesheets (`static/css/`)
- `dashboard.css` - Dashboard page styling
- `login.css` - Login page styling  
- `monitor.css` - Monitoring page styling
- `predict.css` - Prediction page styling

#### JavaScript (`static/js/`)
- `dashboard.js` - Dashboard functionality and charts
- `login.js` - Login form handling
- `monitor.js` - Real-time monitoring and data updates
- `predict.js` - Prediction form and result display
- `register.js` - User registration handling

### HTML Templates (`templates/`)
- `dashboard.html` - Main dashboard view
- `login.html` - Login page
- `monitor.html` - Real-time monitoring interface
- `predict.html` - Prediction interface
- `register.html` - User registration page

## Technology Stack

- **HTML5**: Structure and semantics
- **CSS3**: Styling and responsive design
- **JavaScript (Vanilla)**: Client-side interactivity
- **Chart.js**: Data visualization
- **Fetch API**: Asynchronous data loading

## Integration

The Flask backend is configured to serve these static files and templates automatically using relative paths.
